Alternate Play

Convey simple drill and set play ideas to your team.

Use as a static drawing board, or create a play frame by frame and animate.

Share with your team through link. No log-ins or accounts required.

Controls / How to Use

Frames

Each play consists of up to 10 player positions. These snapshots are equally spaced in time during animation. Set initial positions, then add more frames to complete your play.

Edit / Create Mode

Edit mode allows allows you to edit a player position / highlighting at the current frame only. This mode is useful when modifying an existing play and you're making minor adjustments.

In Create mode, every change at the current frame will modify all subsequent frames. This is the default mode when creating a new play.

Animation

Animations can be done with players moving at the same speed in robotic fashion, or you can choose natural movement in which objects accelerate from the origin, then slow down when approaching their destination.

Saving and sharing your play

Alternate Play doesn't record any plays, and there are no log-ins or usernames, so you are responsible for storage of your plays.
To store or share your play, press "SAVE TO URL". This newly created URL can be used at any time to view the play.

Browser Requirements

This app uses browser features that might not available if you are running Internet Explorer 9 or an earlier version. Similarly, outdated versions of Chrome, Safari, Firefox, etc. might prevent the app from displaying properly. A quick automatic update of your browser should fix any compatibility issues.
